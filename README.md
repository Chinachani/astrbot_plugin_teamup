# 谁tm和我组队

支持多条组队信息（招募大厅），可跨群共享或仅限本群。

## 命令
- /组队菜单
- /组队创建 <每队人数> <组队名> [本群|跨群]
- /组队切换 <组队名> [本群|跨群]
- /组队大厅 [本群|跨群]
- /组队我的 [组队名] [本群|跨群]
- /组队建队 <队伍名> [组队名] [本群|跨群]
- /组队加入 [队伍名] [组队名] [本群|跨群]
- /组队退出 [组队名] [本群|跨群]
- /组队列表 [组队名] [本群|跨群]
- /组队空缺 [组队名] [本群|跨群]
- /组队随机 [组队名] [本群|跨群]
- /组队随机 全部 [组队名] [本群|跨群]
- /组队昵称 <昵称>
- /组队重置 [组队名] [本群|跨群]

## 说明
- 跨群组队为“招募大厅”，所有群可见可操作。
- 本群组队仅本群可见。
- 管理员才能创建、全员随机、重置。
- 群员随机仅在“未组队/队伍仅1人”时可用。
- 跨群列表会在昵称后显示 QQ 号。
